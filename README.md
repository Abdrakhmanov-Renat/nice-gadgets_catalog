# Getting Started

1. Clone the repository:
```bash
  git clone https://github.com/Abdrakhmanov-Renat/nice-gadgets_catalog.git
  cd nice-gadgets_catalog
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
3. Run the project locally:
```bash
npm start
# or
yarn start
```
